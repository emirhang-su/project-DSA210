Description
Prevent script kiddies from plastering your screen with garbage!

WATCH THE VIDEO FOR TUTORIAL!

This mod places your game in a "safe mode" where nothing can display images or play sounds while active.

Using the GUI, you can identify if any mods are currently trying to be malicious, and view recently updated mods.

RECOMMENDED GITHUB VERSION (MORE FEATURES AND SECURITY:
https://github.com/wontairr/GarrysMod-AntiScreamer


Watch the video (2:00 in) for a tutorial on how to download the github version!


YOU *DO NOT* KEEP THIS ENABLED WHILE PLAYING! SEE BELOW USAGE:

Example of Intended Usage

You start up the game with the Anti-Screamer for the first time (github version). Once the stack viewer appears, you refresh, then ignore all addon paths that you trust (HUD mod for example) by right clicking and clicking "Add to Ignore List".

Every time after this now, you check the recently updated addons via the options, and skim over the stack viewer to see if anything is going on. After all of that you simply disable the anti-screamer via the options and continue playing.

In the scenario that you cant find anything in the stack viewer, but there is a hidden screamer in one of your mods, remove all ignored paths in the options->edit ignored paths, then refresh the stack viewer to see any new function calls. Read thoroughly all the function calls and arguments, then assess what the culprit is. Once you do, report it to the top discussion on this page (SCREAMER WATCH).
IF YOU CAN'T SEE COMMENTS AT THE BOTTOM OR LINKS ARE REMOVED, THE MOD IS BEING FALSELY REPORTED.
