Description
G ZEUS Addon inspired by ZEUS mode from ARMA 3

WORKS ONLY IN MULTIPLAYER OR ON DEDICATED SERVERS

Briefly described addon features (Most are shown in gif)
  • Move objects
  • Rotate objects
  • Copy, paste objects
  • Undo, redo actions done with objects
  • All this also applies to welded objects (Duplicates and manually created)
  • Send npc to cursor position
  • Freeze multiple objects
  • Restore original position of objects
  • Delete multiple objects
  • Destroy objects (does not support history)
  • Attach objects to other objects, but not as welded, but the attached object can be at any distance and has no collision
  • New features in context menu and added new context menu when clicking on empty space
  • Find any object (including players) on the map and teleport to it
  • See the health of NPCs, players and props, and also see the armor of players.
  • See objects and their names through walls (Configurable)
  • Customize most things to your taste in client settings

Default controls (Can be changed in settings)
    Main
  • Toggle ZEUS mode — g
  • Send ping — t
  • Delete selected — DEL
  • Break/Kill selected - END
  • Freeze selected — f
  • Reset object orientation — r
  • Toggle flashlight — n
  • Vertical dragging — ALT
  • Rotation — SHIFT

    Movement
  • Forward — w
  • Left — a
  • Backward — s
  • Right — d
  • Up — q
  • Down — z
  • Fast movement — SHIFT

    Clipboard, history and Multi-creation
  • Multi-creation — CTRL
  • CTRL+C copy — c
  • CTRL+V paste — v
  • CTRL+Z undo — z
  • CTRL+Y redo — y
