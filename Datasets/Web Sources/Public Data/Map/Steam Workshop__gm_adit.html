Description
From the creator of Source Shadows
About
This map is an abandoned underground facility. It has a cave with water, some underground sections, and other various rooms.
This map is fully AI noded, and comes with a nav mesh! I find these to be one of the most important things that most maps skip over, so I've made them!

You don't need any games mounted for the map to work.

gm_adit is my first map. I've been working on it for a while and learning hammer along the way. I hope you have fun playing it!
