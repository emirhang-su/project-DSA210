Description
YOU MUST PLAY ON THE 64 BIT BRANCH FOR THIS MAP TO FUNCTION CORRECTLY.

About This Map
Flux is back and better than ever! gm_flux2 is an abstract map that is heavily inspired by Nipper, Shadgrimgrvy, Nico, InvalidVertex, QuantumDylan, and many others. I love exploration maps, especially if it's a little abstract. So I made second one!
This will most likely be my last Garry's Mod map I'll make for a while since I have a lot more things to focus on in my personal life now. (edit: I lied)

Features
  • 50+ unique rooms
  • 50+ different music tracks (disable with music volume in game settings)
  • Almost every room has its own gimmick
  • Custom soundscapes
  • NO CUBEMAPS (i hate working with those)
  • NPC Nodes
  • NAV Mesh

Extras
