Description
A massive suborbital space station looms just above the atmosphere, consuming and honing celestial energy into violent rays of light via space voodoo magic laser to annihilate quaint little farming villages

(they look more beautiful when they are turned into glass).



Map is called gm_stelliferous

To make sure all materials are properly loaded, make sure you are using the chromium branch of gmod (x86-64)

No additional content required (but for real now), although this map is pretty intensive, especially at the catwalk.
If you want, you can remove the env_projectedtexture entities (the most expensive entities) by typing ent_remove_all env_projectedtexture in the console.

Highly recommended to use HDR, as the fake skyboxes and projectedtexture lamps are made/configured with HDR in mind; mat_hdr_level 2
(you can still play in LDR, but features may appear too dark/bright.)



Features
- Navmesh
- Nodes
- HDR
- Cubemaps
- Light bridges, sliding doors
- Teleporters
- Death/damage triggers
- Orbital laser (you need to find 3 keys to activate it)
- Way too many light sources
- low-gravity "space" environment (its a little scuffed)
- Futuristic/Sci-fi aesthetic
- Custom soundscapes

Locations
- Catwalk
- Lodging
- Logistics
- Pool
- Agriculture
- Servers
- Bridges
- Ventilation System

thanks to natko for reuploading Philip Klevestav's scifi textures 12 years ago
https://gamebanana.com/mods/7425
https://gamebanana.com/mods/7432

and also amigang for the assets of this star trek project 18 years ago
https://www.moddb.com/mods/enterprise-warp-5

music on the radio is Unspeakable Roses - Hello Meteor

(to my knowledge), the rest of the assets are from:
IPs owned/licensed by Valve Software
stock assets
custom-made assets (models, sounds, textures)

the map was intended to be much more expansive (and expensive), but i ended up reducing the playable space in a bid to prevent the compiler from running out of memory lol. might end up adding more sections tho, who knows....
