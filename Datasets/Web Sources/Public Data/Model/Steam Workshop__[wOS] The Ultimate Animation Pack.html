Description
YOU'LL NEED THE WOS DYNABASE ADDON FOR THIS PACK TO WORK!

This addon adds a bunch of animations for your Garry's Mod models, mainly dances, emotes and some poses. There are 471 animations in total.

The pack has 4 animation categories:

Fortnite: Emotes/dances from the game and some idle poses (290 animations)
TF2: Mannrobics, Conga and Kazotsky Kick taunts from all 9 classes (28 animations)
MoCap: Motion capture animations from Bomb Rush Cyberfunk (46 animations)
Other: Animations from various other sources (107 animations)

To animate a ragdoll you can use a tool like Easy Animation Tool or Ragdoll Puppeteer then search for sequence names that start with wos_uap_

Example: wos_uap_mocap_breaking

You can also animate your playermodel with the console command wos_uap <anim_name>

Example: wos_uap breaking

Use wos_uap stop to stop the animation.
You can find the complete animation list here: ANIMATION LIST

The animations are from VRCMods[vrcmods.com]
I used Unity and Blender to bake them to a ValveBiped model and import them on Source.

If you encounter any issues, be sure to check the FAQ, ISSUES & HELP
