Description


Gartic Phone but it's GMod

GitHub: https://github.com/figardo/garryphone

F.A.Q
How do I start the game?
Click where it says Sandbox in the bottom right, then select Garry Phone in that list. You can then select any map to start a game of Garry Phone. Once the map has loaded, the game host can press TAB (or whatever is bound to show scoreboard) and click GO! to begin a round.

Are my mods compatible with this?
Custom ragdolls/models and maps should work fine. Beyond that, no promises!

What is this?
Garry Phone is a recreation of Gartic Phone[garticphone.com] (or Broken Picture Telephone) using the mechanics of GMod Sandbox.

Every player starts by writing a word prompt, which is then sent to another player. This player must create a build out of props that best represents the prompt they've been given. Once finished, each build is then sent to a different player, who has to guess what the build is trying to represent. This cycle is repeated until everyone has made a prompt or build for each other.

Once the game has ended, all prompts and builds are gathered into each player's "album", which everyone gets to go through and laugh at.

Replays
At the end of a game, the host will be given the option to save the replay. Replay files are stored in the data/garryphone/ folder, organised into folders corresponding to the map the replay was played on (e.g. gm_construct).

To load a replay file, start a game in the Sandbox gamemode on the same map that you played on and type gp_loadreplay into console. This will show you a menu that you can load replays from.

Maps
Any map should work! If a particular map doesn't work for you, feel free to comment about it.
