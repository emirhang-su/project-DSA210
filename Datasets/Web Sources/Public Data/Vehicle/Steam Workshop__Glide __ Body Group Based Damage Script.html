Description
THIS SCRIPT NEEDS TO BE INSTALLED SO THAT THE DAMAGE WORKS

The script is available for everyone to use it for their vehicles. This script makes it so that depending on where you hit, the bodygroup will change so if you hit your front bumper, your front bumper will be damaged (if you put the bodygroupid properly). This only works if the creator added support for the script and if his model has damaged bodygroups

Originally was supposed to be an exclusive script only for the upcoming NFS Undercover car pack

This isn't coded by me
