Description
A pack of random vehicles I wanted to port to/test on Glide.

Base is required!

Glide // Styled's Vehicle Base is required for this to work!

Tips

- Hold the Toggle headlights key (default is H) to put the Blazer Aqua in Amphibious mode
- Hold the Toggle headlights key (default is H) to put the Deluxo in flight mode
- While the Deluxo is in flight mode, hold the Lean forward/Lean back keys (default is the up/down arrows) to fly up or go down
