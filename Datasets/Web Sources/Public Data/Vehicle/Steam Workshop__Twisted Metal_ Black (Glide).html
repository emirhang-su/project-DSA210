Description
If you're missing textures or vehicles aren’t working — download all the required content!

ADDON

This addon is dedicated to one of the classic entries in the Twisted Metal series — Twisted Metal: Black.
I manually extracted all the models and textures, adapting them for the Source engine.
Now, many will be able to create exciting scenes using these models! Enjoy the battles!

I spent two weeks on this addon. Please respect the author's work and always link to the original if making modifications.

Features:

- Vehicles can be used ONLY in thirdperson (Driver is invisible and all vehicles have only one sit);

- Weapon changing and usage while driving! (Three types available: Machine Gun, Missiles, and Missile Barrage);

- Bodygroups for vehicles;

- Functioning headlights and taillights on all vehicles (except Warthog). Some even have working sirens!;

- Custom collision models (you can even climb inside Sweet Tooth’s truck!);

- Helicopter.;

- Sweet Tooth now can turn into Special one! (Just push the "L" button on keyboard!);

- Simple yet dynamic vehicle physics, with speeds ranging from 130-160 km/h and rapid acceleration for an intense experience.

Addon Authors

Don Juan – Models and texture porting, Lua and fixes;
Ar|es – Tester;
Bot01, Bot02, Bot03 – Legendary testers.
