Description
Notice: NEAD will not be receiving any feature updates anymore. Instead, I plan on rewriting everything in the future and publishing a definitive version of NEAD, which will include base features reworked and additional features at release. Thank you, everyone.



I RECOMMEND YOU WATCH THE VIDEO FOR INFORMATION ON WHAT THIS ADDON DOES. (its kinda goofy, but it does the job)



This Mod Aims to make the way NPCs and DRG nextbots detect you more realistic, not overhauling but rather adding onto the entity's base AI for maximum compatibility. Aside functionality, you shouldnt even see the performance difference between having the mod installed or uninstalled (unless you like playing around with 900 projected textures)



--Features n' Whatnot--

(1) Light Levels Matter

-In the absence of light, you can hide from the eyes of entities. Static lighting, dynamic lighting or light within a projected texture's FOV will reveal you to an entities vision.

-Dynamic lights and projected textures operate in real time, and so does this mod. Any BASE gmod light tools you spawn or map lights will work. (Heads up: some modded light entities such as Advanced Light Entities that use a clientside projected texture wont be detected. Only serverside ones like on maps will.)

(2) Unique Sound Detection

-Entities finally found out how to turn their hearing aids up, and will now be able to hear sounds you make if close enough.

-Not only sounds you make directly can be heard, but also things you interact with that make noise. Aside those things, voice chat is also now heard by entities and will be able to hear you depending on how loud you talk.

(3) Chase & Detection Radius

-If something finds you, they wont loose you so easily if you keep running from them. You will have to run far enough away and hide.

-On top of that, entities will track to the last position they knew you were if they loose you in the dark. Be careful when coming to a stop where you think you can safely hide. (iNPC will kinda change the way this works, NPCs will sort of "decide" if they want to go looking for you or not :\)

(4) Configuration

-Lots of config option in "Utilities/NEAD/Tweakables" for all your tweaking needs.



--Compatibility--

(Works With)

-VJBase

-DrGBase

-Most other NPC bases

-Most weapon silencers

-iNPC (for the most part)

(Doesnt Work With)

-PNG nexbots (because they dont give af if your a no target or not)

-Other nextbot bases than DRG

-DBD nextbots for some screwed up reason



--Goes Good With--

-Stealthy Door Opening: https://steamcommunity.com/sharedfiles/filedetails/?id=2151694266&searchtext=stealthy+door+opening

-Enhanced Proximity Voice Chat: https://steamcommunity.com/sharedfiles/filedetails/?id=3293939372

-Modern Warefare NVGs: https://steamcommunity.com/sharedfiles/filedetails/?id=2532012185&searchtext=Modern+Warefare+NVGs
^This wont work great on multiplayer as the NVGs create a dynamic light on the player, and for some reason only in multiplayer will this reveal you in a dark area.



--Known Issues--

-Open mics probably dont work with this. Just use push to talk if you want this to work.

-Some maps have "wonky" lighting, so those ones will force you to adjust the hell out of Light Sensitivity in the config.

-Dont know if this affects flying non-VJBase NPCs, because someone said it looks like it does. In my experiences it doesnt seem to, they just ignore sounds.

-Other addons that no target the player (like an admin mod that can set players as no target) wont keep the player no targeted. If you wish for this to not interfere with other addons that have a no target function, disable Light Level Spotting in settings.



--MULTIPLAYER--

I tested it with a friend and it worked pretty good. However, its still a little bit WIP so let me know how it does if you play with this in multiplayer. Also, dedicated servers arent supported right now.



Like usual, let me know about errors and stuff, and I'll fix it when I get the chance. Gonna work on a map next.
