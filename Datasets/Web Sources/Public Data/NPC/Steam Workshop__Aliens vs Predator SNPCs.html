Description
Welcome to my Aliens vs Predator SNPCs, one of several mods I've been developing for 3 years now!

For the best performance, please use x64 bit branch of Garry's Mod!

Controller is currently broken for these guys. DrVrej broke it with his last update, he has had a patch ready but hasn't released it for some time. All we can do is wait on him.

This is a very large Aliens vs Predator mod comprising of all the original NPCs/Weapons from AVP3, as well as some custom/additional NPCs based on other AVP games. Each NPC has unique AI that makes them stand out from normal VJ Base mods, including stealth tactics and light detection. Xenomorphs and Predators can perform fatalities on most NPCs in the mod. Xenomorphs exhibit their full life cycle, from ovamorph all the way to Queen caste. Each NPC can be fully controlled by the player with unique HUD elements, viewmodels and controls.

Known Bugs/Issues
  • Spawn placements in survival mode can sometimes put NPCs in bad areas. There are some built in checks to get around this most of the time, but on occasion they will be stuck there
  • Fatalities are not fully finished/fleshed out
  • Combat Android Elites have messed up camo. This is a bug with Source engine itself, where matproxies don't operate correctly on entities with children parented to them
  • (2026) As of a recent Garry's Mod update, Rubat changed something somewhere (I legit have no clue) that broke the spy shader on NPCs, so cloaking looks very wrong for Predators and Combat Android Elites


To Do
