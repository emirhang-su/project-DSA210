Description


Hello there! Welcome to my Black Mesa: Definitive Edition Weapons addon!
(only named it like this so people won't confuse it with other pack)

This addon contains quality ports of weapons from Black Mesa for Garry's Mod(+ 1 addition)

NPC Version

Weapons:
Black Mesa:
Pistol
Crossbow
Crowbar
.357 Magnum
SMG
Rocket Launcher
Тau Cannon
Shotgun
Gluon Gun
Charged Gluon Gun
Hive Hand
Frag Grenade
Satchel Charges
Tripmines
Snarks

Entities:

Black Mesa:
Snark Nest
Buckshot Ammo
MP5 Grenade
.357 Magnum Ammo
RPG Round
Pistol Clip
MP5 Clip
Uranium Box
Crossbow Darts

Known Issue(please read before reporting a bug):
When you first equip Satchel hands will be invisible. To fix this you should respawn and equip it again
Hivehand has a small chance to cause crashes

Features:
C_Hands support!
Cool VFX
Cool SFX
Icons
Configurable with settings
Full Russian and English localizations
Full functionality from the game
Ability to lower weapon(press E + R)
Sprint animations
First draw animations
Prop ejecting
View Punch effect
Dynamic Light effect
Working Iron sights and Scope with all respecful effects
Functional in multiplayer
Gluon Gun has unique world model if it doesn't have owner
Snark is an actual NPC
Kill Icons
Weapon Icons
Category Icon
Crosshairs
Colorable HUD elements
90% of addon is hexed and won't replace anything

Settings:

Tool Menu >> Options >> Limakenori >> Black Mesa Weapons

Enjoy!


Credits:
Crowbar Collective - Black Mesa and assets from it
Limakenori - coding, some .QC, material and texture edits
