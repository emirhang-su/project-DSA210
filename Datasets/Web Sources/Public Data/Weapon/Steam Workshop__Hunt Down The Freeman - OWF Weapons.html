Description
Brings the complete arsenal from the Operation Whisky Freedom build of Hunt Down The Freeman into Garry's Mod. Some of these weapons have either not been added yet or have been discarded.

The weapons included are:
  • M1911
  • AKMS
  • AR2
  • AWP
  • Axe
  • Binoculars
  • Claymore
  • Double Barrel
  • Flare
  • Flare Gun
  • Gas Mask (Protects 100% against nerve gas and 90% against radiation)
  • Glock
  • MK3A2 Grenade
  • USP Match
  • Kar98 (Includes an iron sight variant)
  • Knife
  • M16
  • M4A1
  • M67 Grenade
  • Machete
  • Revolver
  • Minigun
  • Molotov
  • MP5 (Includes variant with suppressor)
  • Night Vision
  • Painkillers
  • M9
  • Remington 870
  • SPAS-12
  • Sig 556
  • MP7
  • Skorpion
  • TMP
  • Synth Grenade
  • Wrench

Main Features:
  • C_hands.
  • Aim down sights.
  • Special attack bind assigned to the mouse wheel button by default (for AR2, M16 and Double Barrel).
  • OWF build crosshairs, sounds and particle effects.
  • NPC usable weapons.
  • Some NPCs can Alt-Fire (e.g., Combine Soldiers, Rebels).
  • Render Target scopes (with customizable resolution).
  • Entities category with ammo and pickups.
  • Customization from the Spawnmenu category "Hunt Down The Freeman" in the "Utilities" tab.
