Description
Stabbing, Impaling for all the default SHARP props!

Have you ever wanted to make...
A working pit of spikes?


Have you ever put sawblades, spikes, all over a car?
What if that...
DID something...


What if spikes could make anything into a weapon?



Anything.





If you aren't yet impaled on SHARPNESS, here's a...
More complete feature list.
  • It works with both serverside, and clientside ragdolls!
  • Although the clientside ragdoll support is a tiny bit 'goofy ahh'.
  • Clientside ragdolls are also, only impaled, if something is killed by a sharp prop.
  • Some sharp props will stick into props, the world!
  • Eg sawblade, harpoon. ( make sure to not throw with gravity gun, that's default )
  • Sharp list is HARDCODED.
  • But it's possible for other mods to register sharp props.
    ( if you, say, want to make melons sharp.... )
  • Since props are added manually, each prop has different characteristics.
  • And hardcoded doesn't mean a short list here.
  • I've added as many props as I could find!
  • Some highlights, the obvious ones, sawblades, trap-propeller-blades, meathooks.
  • And your more obscure hazards, like tables.. chairs.. I-Beam ends...
  • Just, watch your step.
  • And finally, SHARPNESS is built from the ground up for performance, it won't lag your game.
    ( won't lag with keep corpses OFF, at least... )

Commands!
sharpness_sv_proppushdamage ( for servers! gravity gun is fair game, and yes, it works even if they throw the props... ) sharpness_sv_sharpdamagemul ( change how sharp stuff is ) sharpness_sv_nobigdamage ( default off, stops damage numbers from blowing up ) sharpness_sv_debug ( enables some developer 1 visualizers, prints )
