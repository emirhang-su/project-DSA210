Description
Hello, everyone!
Looking for an add-on to work universally on any map (preferably with a navmesh) and creep out you and your friends?



What is this?
This is essentially an add-on that adds random paranormal events to any map with a navmesh.


Current features:
•Ambient sounds play at random intervals around the map.
•Ghosts will spawn and lurk around the map, flinging props, opening doors, etc.
•Shadow Figures will spawn.
•Cockroach swarms will spawn randomly and skitter across the ground.
•Blood will sometimes drip from the ceiling.
•Doors will open and close at random.
•Buttons and switches will be manipulated randomly.
•Lights will flicker at random.
•Lights will break at times.
•Ghost orbs will spawn.
•Highly configurable with convars for setting your preferred options
(Just type gmpa_ and see the list of options!)


Recommended maps:
•M3 Facility
•Amnesia TDD Map
•City of Silence
•Never Lose Hope Hospital Complete
•Granny 3 House
•Backrooms
•The International Mall (no navmesh)
•rp_isidore (no navmesh)
•gm_jammed (no navmesh)


ConVar Name
Description
gmpa_enabled
Enable or disable the ghost events system.
gmpa_max_active_ghosts
Maximum number of ghosts that can be active at once. (default is 5)
gmpa_radius
Radius in hammer units that dictate how far or how close events/apparitions should happen from a player. (2048 default)
gmpa_ghost_lifetime
Maximum time in seconds for a ghost's physical form to linger around. (default is 10 seconds)
gmpa_delay_event_min
Min time in seconds between ghost events (default is 5).
gmpa_delay_event_max
Max time in seconds between ghost events (default is 90).
gmpa_delay_apparition_min
Min time in seconds between ghost apparitions (default is 30).
gmpa_delay_apparition_max
Max time in seconds between ghost events (default is 120).
gmpa_aggro_threshold
Aggro threshold for ghost (default is 10).
If the aggro surpasses this value, the ghost will hunt the player that triggered it.)
gmpa_aggro_multiplier
Aggro multiplier for ghost. (default is 1)
gmpa_ceiling_bleeding
Enable bleeding effects from ceiling? (1 = enabled, 0 = disabled).
gmpa_shadow_figures
Enable shadow figures? (1 = enabled, 0 = disabled).
gmpa_ghost_orbs
Enable ghost orbs? (1 = enabled, 0 = disabled).
gmpa_visible_ghosts
Are ghosts visible? (1 = enabled, 0 = disabled).
gmpa_cockroach_swarms
Enable cockroach swarms? (1 = enabled, 0 = disabled)
gmpa_prop_flinging
Enable prop flinging? (1 = enabled, 0 = disabled)
gmpa_flickering_lights
Enable lights flickering throughout the map? (1 = enabled, 0 = disabled)
gmpa_flashlight_effect
Enable flashlight effects on the ghost system? (1 = enabled, 0 = disabled)
gmpa_flashlight_flicker_chance
Chance of flashlight flickering or turning off when near ghost. (Default is 25, 100 means your flashlight always flickers in close proximity to a ghost.)
