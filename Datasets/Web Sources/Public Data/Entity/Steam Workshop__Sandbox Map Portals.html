Description
Hii! I created a mod to make changing maps more exciting and with flare.

Features
Portals to other dimensions (A.K.A. other maps)
- Easily select and add your own map (or all of it) to the list of portals
- Have your own presets to save!
- Once in the list and restarted, it will create an entity portal for that specific map (Spawnmenu -> entities -> WePortal)
- Spawn it and get in to transport to that map!

Portal Storms
- Too lazy to spawn it yourself? Turn this feature on to dynamically spawn portals everywhere
- Fancy aesthetics as they invade your map
- Will work on any map with or without Nodes and Navmesh (It's still better to have navmesh/node for better placements though :3)

Map/Player Persistence on Portal travel
- Will not reset the player's weapons,health,armor,ammo on portal travel
- Will save the previous map's state when jumping through portals
- Featured can be turned off
- Map/Player will not persist if you shutdown or make a new server

Portal Tools and entities
- Manually spawn in the portals in the entities tab
- Manually spawn in portals using Map Portal Tool (under WeTools)
- Destabilize portals with the right click of Map Portal Tool


How to get started
- Go to Spawnmenu -> Options -> Wetools -> Portal Options
- Select one of the Quick Add buttons (Add All maps, Add All addon maps)
- Select "Add Portal" to add a portal for a specific map
- or Select "Add Portals by name" and type "gm_", in which will add ALL maps that has "gm_" in it.
- Check the checkbox of the "Enable Portal Storms?"
- Restart the server
- Wait for few minutes for portals to invade you

Recommended mods
Black Mesa First-Person Animations - for fancier map hopping
Echoes from Beyond - Works really well once you setup the portals in sandbox.

How:
1. Create a server in sandbox gamemode, setup the portals, turn on portal storms
2. Close the server then change gamemode to Echoes From Beyond (Keep the sandbox Map portals mod enabled)
3. Startup a server on any map
4. Now you can enjoy Echoes from Beyond strolling around and find random portals to hop to other maps without accessing the menu!

Other Recommended mods
Sandbox Collectathon - if you feel like map exploring
Nav Hunt - if you feel like map exploring (I said that again D:)

Prop Shop Progression - if you need to find more props to progress here

Notes
- This was tested in the x84-64 branch, if something might not work try:
Go to your Steam library -> Right click Gmod -> properties -> Betas -> Beta Participation -> Select "x86-64 - Chromium + 64-bit binaries"
- It will be kinda trolly in multiplayer since the whole server will change map
- Turning on persistence feature conflicts with Black Mesa First-Person Animations mod

Got any suggested map collections to try this out? Feel free to share in the discussion, Id like to play with this too XD
