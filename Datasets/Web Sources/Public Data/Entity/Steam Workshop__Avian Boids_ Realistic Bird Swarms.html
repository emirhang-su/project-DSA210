Description
Description
This addon implements a Boids Simulation (Separation, Alignment, Cohesion) combined with a 3D Spatial Partitioning Grid to handle large numbers of birds.

Inspired by the work of Craig Reynolds and Sebastian Lague.

Github
https://github.com/Nayyl/Garry-s-Mod-Avian-Boids-Realistic-Bird-Swarms-Addon

⚠️ Performance Note
This addon is designed for large-scale simulations. However, spawning hundreds of birds is CPU intensive. I've tried to optimize the logic using a grid-based neighbor search.

Key Features
  • Autonomous Flight: Birds avoid obstacles using a pre-computed Fibonacci sphere raycasting system.
  • Flocking Behavior: Realistic group movements (Murmuration).
  • Fully Customizable: Control everything via ConVars (Speed, Detection Range, Factors).
  • Optimized Grid System: Reduces CPU usage by limiting calculations to nearby grid cells.

Future & Feedback
This is my first release, and I plan to evolve it based on your interest:
  • Fish Swarms: If requested, I can develop a dedicated underwater version.
  • More Species: Adding more bird models and specific behaviors.
  • Optimization: I am always looking for ways to improve the code.

I am looking for constructive feedback, If you have any issues, suggestions, or optimization ideas, please leave a comment.

Main ConVars
sv_boids_speed (Default: 600) sv_boids_trace_lengh (Default: 500) sv_boids_cell_size (Default: 1000) sv_boids_separation_factor (Default: 1.5)

Credits
  • Sebastian Lague: For the brilliant explanation of boids and optimization (https://github.com/SebLague/Boids).
  • Craig Reynolds: The father of the Boids algorithm.
