Description
What is this?
This addon brings Roblox-style 3D chat bubbles to GMOD. Whenever a player sends a message, a floating text bubble appears above their head just like in Roblox

Features
  • Lambda Players Support
  • 3D2D chat bubbles with automatic text wrapping and multiple lines
  • Distance-based visibility and simplified display when players are far away
  • Full admin configuration menu (bubble limit, duration, anti-spam, slowmode, toggle GMod default chat)
  • Optional censorship system with customizable blacklist
