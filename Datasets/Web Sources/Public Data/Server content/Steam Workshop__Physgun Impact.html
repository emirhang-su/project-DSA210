Description
Physgun impact is an addon made to give more immersion to the physic gun.

Ever got frustrated by not being able to crush/destroy an NPC or a player, by using the beam of the physic gun? Now you can!

Physgun impact allows you to crush/destroy anything you want while using the physgun force! You dont need to throw the prop/entity anymore! Want to kill a zombie? Just smash him using the physgun!

Physgun Impact was tricky to make but I thought of a way to make it work. The way it works is really interesting, it takes the speed of the prop you hold with your physics gun, there is a speed threshold, if the speed goes over the threshold, or is right in the threshold, it will do damage, working as a camouflaged physics system.
(Keep in mind that the way physics/speed work for the Source Engine is quite complex, please change the speed threshold if needed.)


This addon also includes:
Convars
-physgunimpact_cooldown
-physgunimpact_multiplier
-physgunimpact_speed
Settings panel to change variables. (Utilities-Physgun Impact-Settings)
Immersion.
Physgun interaction.

Also, I suggest not playing with the variables of cooldown and multiplier, I made the default the best I could to make it feel good yet balanced?
You are free to play around with the speed threshold.

This is experimental, if you find any bugs or any problems, report them, as for a quick fix, you may need to quirk the speed threshold depending on the problem. NPC does not die? Try to lower the speed threshold and viceversa.

TESTED ALONE.

If you find/know a similar addon to this please inform me!
