Description
What is this?
Simple script that adds spawn protection for all players

Features
  • Customizable protection duration
  • Toggable bubble effect indicating spawn protection status
  • Toggable chat notifications about spawn protection status
  • Option to prevent protected players from dealing damage
  • Optional NPC ignore setting, NPCs won’t attack protected players

For developers

You can use `ply:IsPlayerSpawnProtected()` to check if a player has active spawn protection
